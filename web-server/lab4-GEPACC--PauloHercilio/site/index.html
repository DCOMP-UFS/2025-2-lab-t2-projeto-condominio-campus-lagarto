Sobre o Laboratório

O laboratório do GEPACC (Grupo de Estudo e Pesquisa em Acessibilidade) é um espaço dedicado ao desenvolvimento de pesquisas voltadas à inclusão digital e tecnologias assistivas.

Nossa infraestrutura de rede foi projetada para suportar:

  • Desenvolvimento e teste de aplicações web acessíveis
  • Estudos de usabilidade com diferentes perfis de usuários
  • Prototipação de tecnologias assistivas
  • Colaboração remota com outros grupos de pesquisa

Infraestrutura de Rede

Conectividade

  • Rede cabeada Cat6
  • Wi-Fi de alta velocidade
  • Gateway dedicado
  • Switch gerenciável L2

Servidores

  • Servidor Web
  • Banco de Dados
  • Servidor de Aplicação
  • Autenticação centralizada

Segurança

  • Nobreak 3kVA
  • Rack organizado
  • Controle de acesso
  • Backup automatizado

Linhas de Pesquisa

Acessibilidade Web

Desenvolvimento de sites e aplicações seguindo diretrizes WCAG, garantindo acesso universal à informação.

Tecnologias Assistivas

Criação e adaptação de ferramentas que auxiliam pessoas com deficiência no uso de computadores e dispositivos.

Usabilidade e UX

Estudos sobre experiência do usuário com foco em interfaces inclusivas e de fácil utilização.

Inclusão Digital

Projetos voltados à democratização do acesso às tecnologias da informação e comunicação.
